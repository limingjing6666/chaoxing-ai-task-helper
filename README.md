# 超星 AI 助手

一键完成超星学习通的 **AI 实践对话** 和 **主题讨论** 任务，支持批量操作和自动重试冲分。

> 从 GitHub 下载或克隆后，浏览器加载扩展时请选择 `plug-core` 文件夹，不要直接选择仓库根目录。

---

## ✨ 能帮你做什么

- **AI 实践**：自动和 AI 老师完成多轮对话，提交后自动获取评分
- **主题讨论**：自动阅读话题，生成有深度的回复并发表
- **批量处理**：一次扫描整门课的所有任务，勾选想做的，全自动逐个完成
- **自动冲分**：设定目标分数，不达标自动重来，越来越好

---

## 📦 安装

支持 **Chrome**、**Edge** 等主流浏览器。

如果你是从 GitHub 获取项目，请先进入仓库里的 `plug-core` 目录，再按下面步骤加载扩展。

1. 浏览器地址栏输入 `chrome://extensions/`（Edge 输入 `edge://extensions/`）
2. 打开右上角的 **开发者模式** 开关
3. 点击 **加载已解压的扩展程序**，选择 `plug-core` 文件夹
4. 完成！打开超星学习通相关页面，右侧会自动出现控制面板

---

## 🔑 准备 API Key

本插件使用 DeepSeek AI 生成回复，你需要一个 API Key：

1. 打开 [platform.deepseek.com](https://platform.deepseek.com)，注册并登录
2. 进入「API Keys」页面，点击创建，复制生成的密钥（以 `sk-` 开头）
3. 回到插件面板，粘贴到 **API Key** 输入框，点击 **保存**
4. 点 **测试连接** 确认可用

> 🔒 你的 API Key 只保存在浏览器本地，不会被上传到任何地方。

## ⚙ 设置

打开任意超星任务页面，展开面板中的 **⚙ 系统配置_SYS**：

| 设置项 | 怎么填 |
|--------|--------|
| **API Key** | 粘贴你的 DeepSeek 密钥 |
| **接口地址** | 不用改，默认就行 |
| **模型** | `deepseek-chat` 速度快；`deepseek-reasoner` 质量更高但稍慢 |
| **轮数** | 对话几轮，建议 5～8 轮 |
| **间隔** | 每轮等待秒数，建议 3 秒以上 |
| **对话风格** | 选一个你喜欢的风格（见下方） |
| **自定义提示词** | 可选，额外补充要求 |
| **目标分数** | 填 0 表示只做一次；填 85 表示不到 85 分自动重来 |

设置好后点 **保存** 即可，所有页面共享同一套配置。

### 四种对话风格

| 风格 | 适合场景 |
|------|----------|
| 🎯 **专业均衡** | 通用，大多数任务选这个就好 |
| 🔬 **技术探讨** | 偏理论类任务，喜欢深挖原理 |
| 🛠 **实战导向** | 偏实操类任务，强调动手经验 |
| ⚡ **精准高效** | 希望回复简短精炼 |

## 🚀 使用方法

### 一、完成单个 AI 实践

1. 在超星学习通打开一个 **AI 实践** 任务
2. 右侧面板自动显示任务信息
3. 点击 **▶ 开始任务**
4. 等待自动对话完成，面板显示最终评分

**想要更高分？** 在设置里填写 **目标分数**（比如 90），没达到会自动重做，每次都比上次更好。

### 二、完成单个主题讨论

1. 在超星学习通打开一个 **主题讨论** 页面
2. 右侧面板自动识别为讨论任务
3. 点击 **✏ 自动回复**
4. AI 读取话题内容，生成回复并自动发表

### 三、批量完成整门课

这是最省心的方式，一次搞定所有任务：

1. 进入课程的 **任务列表页**
2. 面板显示 **批量任务** 区域，点击 **📡 扫描任务**
3. 扫描完成后列出所有任务，带有类型标签：
   - 🤖 **AI** = AI 实践任务
   - 💬 **讨论** = 主题讨论任务
4. 每个任务左侧有 **勾选标记**，点击可选择或取消
5. 点 **全选 / 取消全选** 快速切换
6. 点击 **🚀 批量执行**，插件会自动逐个打开、完成、跳转下一个

**中途想停？** 随时点 **■ 中止** 即可。如果关了浏览器，下次打开会提示你是否继续上次的批量任务。

## 💡 高分技巧

- **轮数设 5～8 轮**，对话越充分评分越高
- **选 `deepseek-reasoner` 模型**，回复质量更好（但会慢一些）
- **风格选「技术探讨」或「实战导向」**，这两种通常得分更高
- **目标分数建议设 85～90**，太高会反复重试消耗额度
- **自定义提示词**可以针对具体任务加料，比如"重点讨论 XXX 方面"

## ❓ 常见问题

**页面上看不到插件面板？**
确认插件已启用。面板只在 AI 实践、主题讨论、课程任务列表这三种页面出现。

**提示"作答次数已用尽"？**
这个任务没有剩余机会了，需要等老师重置次数。

**提示"当前任务已提交"？**
点页面上的"重新练习"新建一次作答，或者设置目标分数让插件自动处理。

**批量扫描没找到任务？**
确认你在正确的课程任务列表页。如果课程没有 AI 实践或讨论任务，就扫不到。

**讨论任务打开后报错？**
尝试重新扫描任务列表，插件会自动获取正确的讨论页地址。

**提示网络连接失败？**
检查 API Key 是否正确填写并保存。点 **测试连接** 确认可用。

**API Key 会泄露吗？**
不会。密钥只保存在你的浏览器本地，请求直接发给 DeepSeek 官方，不经过任何第三方。
